为什么要写作
那么我坚持做下去。
一方面能缓解，现实给心情带来的劳损。
另一方面，还能坚持自己的信念以及活络自己的思路，还有振奋情绪。

在记录生活，调整认识。整理情绪等方面，对我都有很大的作用。

在现阶段，明确自己这个时期的生活，生存的主题。
持续优化，完整自我。以及取得成就。
这是我最看重的东西。

以我现在的人生，不足以支撑我实现我的目标
也许，等变得不需要文字的支持，我也可以实现我自己的目标。
我就不需要这个东西了。

没办法，我没有一个良好的生存，发展的环境。
没事，我自己建立一个。
